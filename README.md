# 🎬 CineSage

CineSage is a Python-based application for exploring and interacting with movie-related information. The project is organized into modular components for the core logic, user interface, and application entry point.

## Features

- 🎥 Movie information lookup
- 🧠 Modular Python architecture
- 🖥️ Simple user interface
- ⚡ Easy to extend with new features

## Installation

1. Clone the repository:

```bash
git clone [https://github.com/YOUR_USERNAME/cinesage.git](https://github.com/shivansh4565/Cinesage_project)
cd cinesage
```

2. Create a virtual environment (optional):

```bash
python -m venv .venv
```

3. Activate the virtual environment:

**Windows**
```bash
.venv\Scripts\activate
```

**Linux/macOS**
```bash
source .venv/bin/activate
```

4. Install dependencies:

```bash
uv sync
```

or

```bash
pip install -r requirements.txt
```

## Running the Project

```bash
python main.py
```
